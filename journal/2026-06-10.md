Like most people around the world today was the first time I was able to access
Fable - Anthropic's Mythos class model designed for general use. While I was
only able to use it on a few different tasks focusing on text processing between
meetings I found the models represent an improvement over what I've been doing
with skills using Opus 4.x.

I think what we're seeing is the leveling of the S-curve when it comes to AI
capabilities. From the news I had heard about Mythos I was expecting an almost
omniscient agent that is able to reason and automatically work through a task.
What I found was an agent with all the same problems that Opus and Sonnet have.
I had given it a design problem that required it to copy a `png` into an
architecture diagram and it struggled to understand what to do.

It will be interesting to see if Fable is worth the 2x token cost over Opus -
something tells me that the cost-to-value ratio won't stack up for a lot of
tasks.
